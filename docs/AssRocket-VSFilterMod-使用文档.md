## 颜色和透明度标签

### 基础颜色标签

#### `\1c&H<bbggrr>&` - 主要颜色
设置文字主体颜色（BGR格式）
- **参数**: 十六进制颜色值
- **示例**: `\1c&H0000FF&` (红色)

#### `\2c&H<bbggrr>&` - 次要颜色
设置卡拉OK效果的次要颜色
- **参数**: 十六进制颜色值
- **示例**: `\2c&H00FF00&` (绿色)

#### `\3c&H<bbggrr>&` - 边框颜色
设置文字边框颜色
- **参数**: 十六进制颜色值
- **示例**: `\3c&HFFFFFF&` (白色)

#### `\4c&H<bbggrr>&` - 阴影颜色
设置文字阴影颜色
- **参数**: 十六进制颜色值
- **示例**: `\4c&H000000&` (黑色)

#### `\c&H<bbggrr>&` - 快捷主要颜色
等同于 `\1c`，设置主要颜色
- **参数**: 十六进制颜色值

### 基础透明度标签

#### `\1a&H<aa>&` - 主要透明度
设置文字主体透明度
- **参数**: 00(不透明) - FF(完全透明)
- **示例**: `\1a&H00&` (完全不透明)

#### `\2a&H<aa>&` - 次要透明度
设置卡拉OK效果的次要透明度
- **参数**: 00(不透明) - FF(完全透明)

#### `\3a&H<aa>&` - 边框透明度
设置文字边框透明度
- **参数**: 00(不透明) - FF(完全透明)

#### `\4a&H<aa>&` - 阴影透明度
设置文字阴影透明度
- **参数**: 00(不透明) - FF(完全透明)

#### `\alpha&H<aa>&` - 全局透明度
同时设置所有4个透明度通道
- **参数**: 00(不透明) - FF(完全透明)
- **示例**: `\alpha&H80&` (半透明)

## 位置和移动标签

### 位置标签

#### `\pos(<x>,<y>)` - 固定位置
设置字幕的固定位置
- **参数**: x, y 坐标
- **示例**: `\pos(320,240)`
- **VSMod扩展**: `\pos(<x>,<y>,<z>)` 支持Z轴坐标

#### `\an<n>` - 对齐方式（小键盘布局）
设置字幕对齐方式（1-9，按小键盘布局）
- **参数**: 
  - 1=左下, 2=中下, 3=右下
  - 4=左中, 5=居中, 6=右中
  - 7=左上, 8=中上, 9=右上
- **示例**: `\an5` (居中)

#### `\a<n>` - 对齐方式（传统）
传统对齐方式（1-11）
- **参数**: 1-11的数字

### 移动标签

#### `\move(<x1>,<y1>,<x2>,<y2>[,<t1>,<t2>])` - 线性移动
从位置1移动到位置2
- **参数**: 
  - x1,y1: 起始坐标
  - x2,y2: 结束坐标
  - t1,t2: 可选的时间范围（毫秒）
- **示例**: `\move(0,0,100,100,0,1000)`

#### `\org(<x>,<y>)` - 旋转中心
设置旋转和缩放的中心点
- **参数**: x, y 坐标
- **示例**: `\org(320,240)`
- **VSMod扩展**: `\org(<x1>,<y1>,<x2>,<y2>[,<t1>,<t2>])` 支持移动旋转中心

## 字体样式标签

### 字体基础

#### `\fn<name>` - 字体名称
设置字体名称
- **参数**: 字体名称字符串
- **示例**: `\fnArial`

#### `\fs<size>` - 字体大小
设置字体大小
- **参数**: 数字（像素）或 +/- 相对值
- **示例**: `\fs48` 或 `\fs+10`

#### `\fscx<percent>` - 水平缩放
设置字体水平缩放比例
- **参数**: 百分比（100=正常）
- **示例**: `\fscx150` (150%)

#### `\fscy<percent>` - 垂直缩放
设置字体垂直缩放比例
- **参数**: 百分比（100=正常）
- **示例**: `\fscy150` (150%)

#### `\fsc<percent>` - 统一缩放
同时设置水平和垂直缩放
- **参数**: 百分比（100=正常）
- **示例**: `\fsc120` (120%)

#### `\fsp<pixels>` - 字符间距
设置字符之间的间距
- **参数**: 像素值（可为负）
- **示例**: `\fsp5`

### 字体样式

#### `\b<weight>` - 粗体
设置字体粗细
- **参数**: 
  - 0 = 正常
  - 1 = 粗体

- **示例**: `\b1` 或 `\b700`

#### `\i<0|1>` - 斜体
设置斜体
- **参数**: 0=关闭, 1=开启
- **示例**: `\i1`

#### `\u<0|1>` - 下划线
设置下划线
- **参数**: 0=关闭, 1=开启
- **示例**: `\u1`

#### `\s<0|1>` - 删除线
设置删除线
- **参数**: 0=关闭, 1=开启
- **示例**: `\s1`

### 字体旋转

#### `\frx<degrees>` - X轴旋转
绕X轴旋转（俯仰）
- **参数**: 角度值
- **示例**: `\frx30`

#### `\fry<degrees>` - Y轴旋转
绕Y轴旋转（偏航）
- **参数**: 角度值
- **示例**: `\fry45`

#### `\frz<degrees>` 或 `\fr<degrees>` - Z轴旋转
绕Z轴旋转（平面旋转）
- **参数**: 角度值
- **示例**: `\frz90`

### 字体变形

#### `\fax<factor>` - X轴倾斜
设置X轴倾斜（水平剪切）
- **参数**: 倾斜系数
- **示例**: `\fax0.5`

#### `\fay<factor>` - Y轴倾斜
设置Y轴倾斜（垂直剪切）
- **参数**: 倾斜系数
- **示例**: `\fay0.3`

#### `\fe<charset>` - 字符集
设置字符集编码
- **参数**: 字符集编号

定义矩形裁剪区域，只显示区域内的内容

#### `\iclip(<scale>,<drawing>)` - 反向矢量裁剪

## 卡拉OK标签

### 基础卡拉OK

#### `\k<duration>` - 标准卡拉OK
标准卡拉OK效果
- **参数**: 持续时间（厘秒，1/100秒）
- **示例**: `\k50` (0.5秒)

#### `\K<duration>` 或 `\kf<duration>` - 填充卡拉OK
从左到右填充的卡拉OK效果
- **参数**: 持续时间（厘秒）
- **示例**: `\kf100`

#### `\ko<duration>` - 边框卡拉OK
边框变化的卡拉OK效果
- **参数**: 持续时间（厘秒）
- **示例**: `\ko75`

#### `\kt<time>` - 卡拉OK起始时间
设置卡拉OK效果的起始时间
- **参数**: 时间（厘秒）
- **示例**: `\kt0`

#### `\kk(<charcount>[,<d1>[,<d2>...]])` - 打字机效果
按字符延时显示后续文本
- **参数**:
  - `charcount`: 受控字符数
  - `d1...`: 每个字符的显示间隔（厘秒），空参数默认 `100`
- **示例**:
  - `\kk(5)`：后续 5 个字符按 1 秒间隔依次显示
  - `\kk(5,150,,200)`：第 1 个字符 150cs，第 2 个字符默认 100cs，第 3 个字符 200cs
- **说明**:
  - `\h` 会计入 `charcount`
  - `\N` / `\n` 不计入 `charcount`
  - 超出 `charcount` 的剩余文本，会在最后一个受控字符显示时一起出现

## 动画标签

### 淡入淡出

#### `\fad(<t1>,<t2>)` - 简单淡入淡出
设置淡入和淡出时间
- **参数**: 
  - t1: 淡入时间（毫秒）
  - t2: 淡出时间（毫秒）
- **示例**: `\fad(300,300)`

#### `\fade(<a1>,<a2>,<a3>,<t1>,<t2>,<t3>,<t4>)` - 复杂淡入淡出
精确控制淡入淡出过程
- **参数**: 
  - a1,a2,a3: 透明度值
  - t1,t2,t3,t4: 时间点（毫秒）
- **示例**: `\fade(255,0,255,0,500,2500,3000)`

### 动画变换

#### `\t([<t1>,<t2>,][<accel>,]<tags>)` - 动画变换
在指定时间内平滑变换标签值
- **参数**: 
  - t1,t2: 可选的时间范围（毫秒）

  - tags: 要变换的标签
- **示例**: 
  - `\t(\frz360)` - 旋转360度
  - `\t(0,1000,\fscx150)` - 1秒内放大到150%
  - `\t(0,1000,2,\fs50)` - 加速变化字体大小

## 其他标签

### 绘图模式

#### `\p<n>` - 绘图模式
切换到绘图模式
- **参数**: 
  - 0 = 文本模式
  - 1 = 标准绘图模式（推荐）
  - 2-n = 缩小绘图模式（数字越大，图形越小）
- **示例**: `\p1`
- **详细说明**:
  - `\p1`: 标准大小，缩放系数 = 1 / (2^0) = 1
  - `\p2`: 缩小一半，缩放系数 = 1 / (2^1) = 0.5
  - `\p3`: 缩小到1/4，缩放系数 = 1 / (2^2) = 0.25
  - `\p4`: 缩小到1/8，缩放系数 = 1 / (2^3) = 0.125
  - 以此类推...
- **注意**: 数字越大，绘制的图形越小，而不是精度越高！所有模式的坐标精度都是相同的（6位定点数，即1/64像素）

#### `\pbo<offset>` - 绘图基线偏移
设置绘图的基线偏移
- **参数**: 偏移像素值
- **示例**: `\pbo10`

### 绘图模式技术细节

#### `\p<n>` 参数的真实含义

很多人误以为 `\p` 后面的数字控制精度，但实际上它控制的是**缩放比例**。

**缩放公式**：
```
实际缩放系数 = 1 / (2^(n-1))
```

**各级别对比**：
| 参数 | 缩放系数 | 实际大小 | 说明 |
|------|----------|----------|------|
| `\p0` | - | - | 文本模式 |
| `\p1` | 1 / 2^0 = 1.0 | 100% | 标准大小（推荐） |
| `\p2` | 1 / 2^1 = 0.5 | 50% | 缩小一半 |
| `\p3` | 1 / 2^2 = 0.25 | 25% | 缩小到1/4 |
| `\p4` | 1 / 2^3 = 0.125 | 12.5% | 缩小到1/8 |
| `\p8` | 1 / 2^7 = 0.0078 | 0.78% | 极小 |

**坐标精度**：
- 所有绘图模式的坐标精度都是**相同的**
- 内部使用6位定点数表示：`坐标值 × 64`
- 理论精度：1/64 像素 ≈ 0.015625 像素
- 与 `\p` 的数字参数**无关**

**实际应用**：
```
// 绘制相同大小的矩形，需要调整坐标值

// 使用 \p1（标准）
{\p1}m 0 0 l 100 0 100 100 0 100{\p0}

// 使用 \p2（缩小一半），需要坐标×2才能得到相同大小
{\p2}m 0 0 l 200 0 200 200 0 200{\p0}

// 使用 \p3（缩小到1/4），需要坐标×4
{\p3}m 0 0 l 400 0 400 400 0 400{\p0}
```

**为什么存在这个参数？**
- 历史原因：早期用于处理不同分辨率
- 实际用途：可以用更大的坐标值绘制小图形，避免精度损失
- 建议：大多数情况下使用 `\p1` 即可

### 渐变颜色标签 (patch m004)

#### `\1vc(<c1>,<c2>,<c3>,<c4>)` - 主要渐变颜色

- **⚠️ 冲突**: 与 `\c` 和 `\1c` 冲突，不能同时使用

#### `\2vc(<c1>,<c2>,<c3>,<c4>)` - 次要渐变颜色
设置次要文字的四角渐变颜色
- **参数**: 四个角的颜色值
- **⚠️ 冲突**: 与 `\2c` 冲突

#### `\3vc(<c1>,<c2>,<c3>,<c4>)` - 边框渐变颜色
设置边框的四角渐变颜色
- **参数**: 四个角的颜色值
- **⚠️ 冲突**: 与 `\3c` 冲突

#### `\4vc(<c1>,<c2>,<c3>,<c4>)` - 阴影渐变颜色
设置阴影的四角渐变颜色
- **参数**: 四个角的颜色值
- **⚠️ 冲突**: 与 `\4c` 冲突

### 渐变透明度标签 (patch m004)

设置次要文字的四角渐变透明度

### PNG背景图像标签 (patch m010)

#### `\1img(<file>[,<xoffset>,<yoffset>])` - 主要文字图像
使用PNG图像作为主要文字纹理
- **参数**: 
  - file: PNG文件路径
  - xoffset, yoffset: 可选的偏移量
- **示例**: `\1img(texture.png,0,0)`

#### `\2img(<file>[,<xoffset>,<yoffset>])` - 次要文字图像
使用PNG图像作为次要文字纹理
- **参数**: 同 `\1img`

#### `\3img(<file>[,<xoffset>,<yoffset>])` - 边框图像
使用PNG图像作为边框纹理
- **参数**: 同 `\1img`

#### `\4img(<file>[,<xoffset>,<yoffset>])` - 阴影图像

## 冲突标签说明

### ⚠️ 重要：渐变标签与普通标签的冲突

VSFilterMod中的渐变标签（`\1-4vc`、`\1-4va`）与普通颜色/透明度标签存在**模式冲突**，不能在同一样式中混用。

### 冲突组1：颜色标签

**渐变颜色标签**（启用渐变模式）：
- `\1vc` - 主要文字渐变颜色
- `\2vc` - 次要文字渐变颜色
- `\3vc` - 边框渐变颜色
- `\4vc` - 阴影渐变颜色

**普通颜色标签**（普通模式）：
- `\c` / `\1c` - 主要文字颜色
- `\2c` - 次要文字颜色
- `\3c` - 边框颜色
- `\4c` - 阴影颜色

**冲突原因**：
- 渐变标签会设置 `mod_grad.mode[i] = 1`（渐变模式）
- 普通标签会设置 `mod_grad.mode[i] = 0`（普通模式）
- 后使用的标签会覆盖前面的模式设置

### 冲突组2：透明度标签

**渐变透明度标签**（启用渐变模式）：
- `\1va` - 主要文字渐变透明度
- `\2va` - 次要文字渐变透明度
- `\3va` - 边框渐变透明度
- `\4va` - 阴影渐变透明度

**普通透明度标签**（普通模式）：
- `\1a` - 主要文字透明度
- `\2a` - 次要文字透明度
- `\3a` - 边框透明度
- `\4a` - 阴影透明度
- `\alpha` - 全局透明度

**冲突原因**：同颜色标签

### 特殊联动机制

当使用 `\1vc`、`\1va`、`\2vc` 或 `\2va` 时，会同时激活主要文字和次要文字的渐变模式：
```
style.mod_grad.mode[0] = style.mod_grad.mode[1] = 1;
```

这意味着设置主要文字的渐变会影响次要文字的渲染模式。

### 正确使用方法

#### ✅ 正确示例1：只使用普通标签
```
{\c&H0000FF&\1a&H00&}文字
```

#### ✅ 正确示例2：只使用渐变标签
```
{\1vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)\1va(&H00&,&H80&,&H80&,&HFF&)}文字
```

#### ❌ 错误示例1：混用导致冲突
```
{\1vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)\c&H0000FF&}文字
```
**问题**：`\c` 会将模式重置为普通模式，渐变设置失效

#### ❌ 错误示例2：混用导致冲突
```
{\c&H0000FF&\1vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)}文字
```
**问题**：`\1vc` 会覆盖 `\c` 的设置并切换到渐变模式

### 在动画中的注意事项

#### ✅ 正确：在 `\t` 中保持一致
```
{\1vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)\t(\1vc(&H00FF00&,&H0000FF&,&HFFFF00&,&HFF0000&))}文字
```

#### ❌ 错误：在 `\t` 中切换模式
```
{\1vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)\t(\c&H0000FF&)}文字
```

### 解决方案

如果需要在同一字幕中使用不同的颜色模式：

1. **使用多个字幕层**：将需要渐变的文字和普通文字分开到不同的字幕行
2. **使用样式重置**：通过 `\r` 标签切换到不同的预定义样式
3. **选择一种模式**：在整个项目中统一使用渐变或普通模式

## 绘图命令

在绘图模式（`\p1` 或更高）下可以使用以下绘图命令：

### 基础绘图命令

- `m <x> <y>` - 移动到坐标（开始新路径，自动闭合）
- `n <x> <y>` - 移动到坐标（不闭合前一路径）
- `l <x> <y>` - 画直线到坐标
- `b <x1> <y1> <x2> <y2> <x3> <y3>` - 三次贝塞尔曲线（3个控制点）
- `s <x1> <y1> <x2> <y2> ... <xn> <yn>` - B样条曲线（多个控制点）
- `p <x> <y>` - 延伸B样条（添加控制点）
- `c` - 显式闭合当前路径

### 坐标系统说明

- 坐标单位：像素（在 `\p1` 模式下）
- 原点：相对于字幕位置
- 可以使用小数：`m 10.5 20.3`
- 内部精度：1/64 像素


#### 三角形（使用 n 命令不闭合）

#### 心形（使用贝塞尔曲线）

## 常见问题

### Q1: 为什么我的渐变颜色不显示？
**A**: 检查是否同时使用了 `\c` 或 `\1c` 等普通颜色标签。渐变标签（`\1vc`等）与普通颜色标签冲突，只能选择一种使用。

### Q2: 如何创建彩虹渐变效果？
**A**: 使用 `\1vc` 设置四个角的不同颜色：
```
{\1vc(&H0000FF&,&H00FF00&,&HFF0000&,&HFFFF00&)}彩虹文字
```

### Q3: 动画不流畅怎么办？
**A**: 
- 减少同时使用的特效数量
- 降低模糊强度
- 使用更简单的绘图路径

### Q4: 如何让文字沿曲线移动？
**A**: 使用 `\moves3` 或 `\moves4` 标签定义样条曲线路径。

### Q5: PNG图像纹理不显示？
**A**: 
- 确认PNG文件路径正确
- 检查PNG文件格式是否支持
- 确保使用了正确的 `\1img` 到 `\7img` 标签

### Q6: 如何实现文字描边渐变？
**A**: 使用 `\3vc` 设置边框的渐变颜色：
```
{\bord3\3vc(&HFF0000&,&H00FF00&,&H0000FF&,&HFFFF00&)}渐变描边
```

### Q7: Z轴效果不明显？
**A**: 
- 增大 `\z` 的值
- 配合 `\frx` 或 `\fry` 旋转查看3D效果
- 确保没有启用 `\ortho1`（正交投影）

### Q8: `\p2` 和 `\p1` 有什么区别？精度更高吗？
**A**: 
- **不是精度问题**！`\p2` 会让图形缩小到原来的50%
- 所有 `\p` 模式的坐标精度都相同（1/64像素）
- `\p<n>` 的缩放系数 = 1 / 2^(n-1)
- 大多数情况下使用 `\p1` 即可
- 如果需要绘制小图形但想用大坐标值（避免小数），可以用 `\p2` 或更高

### Q9: 为什么我的绘图比预期小很多？
**A**: 
- 检查是否使用了 `\p2` 或更高的值
- `\p2` 会缩小到50%，`\p3` 缩小到25%，以此类推
- 要么改用 `\p1`，要么将所有坐标值乘以相应倍数
